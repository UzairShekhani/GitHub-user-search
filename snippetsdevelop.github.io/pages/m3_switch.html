Customize checkbox toggle switch | Google Material 3 design

A toggle switch is a simple yet effective user interface element that allows users to switch between two states (i.e. on and off state). Below is a design for the customized toggle switch that is inspired from the Google Material Desing 3 components. Instead of using complex JavaScript or jQuery, we will be utilizing the simple HTML checkbox element to achieve the desired functionality.
